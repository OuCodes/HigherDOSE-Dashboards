# Tiers

## Tier 1 Metrics
1d, 1st time metrics track true first time customers. 
- Spend 
- CAC (1d, 1st time)
- ROAS (1d, 1st time)
- AOV (1d, 1st time)
- ECR (1d, 1st time)
- eCPNV (1d, 1st time)

## Tier 2 Metrics
1d metrics track first time and existing customers. 
- Spend 
- CAC (1d)
- ROAS (1d)
- AOV (1d)
- ECR (1d)
- eCPNV (1d)

## Tier 3 Metrics

- Spend 
- Platform-reported CAC
- Platform-reported ROAS
- New Customer Percentage
- Revenue Per Visit
